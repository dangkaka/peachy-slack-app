# peachy-slack-bot

A slack app to get random image from instagram by tag or profile
Default will get image from "peachy" profiles :">

## Deployment

Using [apex/up](https://github.com/apex/up)
* Run `up`
* Get the endpoint url by `up url`
* Setup slack application
